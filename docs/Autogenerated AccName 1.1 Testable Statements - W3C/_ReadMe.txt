This folder includes individual tests based on the test cases documented at
https://www.w3.org/wiki/AccName_1.1_Testable_Statements

These files are automatically generated by running the batch file in the root folder "Generate W3C Testable Statements", named "StartGeneratingStatements.bat".

The index.html file within this folder is automatically created to provide direct links to all of these files for testing on a webserver.
